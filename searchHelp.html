How to use Search

The purpose of the Search is to allow the user to search a folder, or folders, that contains DICOM files.

There are 3 search criteria that can be used; these are:

  • Patient Name
  • Body Part Examined
  • Acquisition Date

Starting the search:-

By default all the search options above are disabled; to enable the search, check the box beside the item for which you wish to search.
For instance; to perform a search for "Patient Name", check the box beside it. The search areas for "First Name" and "Last Name" are now enabled and you can type the search criteria into them.

You can perform wilcard searches by entering the following symbols:
Note: These examples are for "First Name", you can use the same criteria for "Last Name"

  • Joe*
    • this will return a list of patients where the First Name starts with "Joe"
  • *mith
    • this will return a list of patients where the First Name ends with "mith"
  • *mac*
    • this will return a list of patients where the letters "mac" are somewhere in the First Name
  • Br?an
    • this will return a list of patients where the First Name is "Brian" or "Bryan"; the ? symbol replaces one letter only

Actions

  • Search
    • Performs the search based on the criteria entered. If no criteria is entered an alert will be displayed
  • Close
    • Closes the search screen and returns to the main DICOM application
  • Help
    • Displays this help screen
